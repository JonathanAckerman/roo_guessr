# Location format

Each playable location lives in its own kebab-case directory:

```text
src/locations/radiant-secret-shop/
  location.json
  question.webp
```

`question.webp` is the full Easy-mode image. Medium and Hard are generated by
cropping around `cropCenter` at runtime, so do not submit separate difficulty
images.

The metadata format is:

```json
{
  "id": "radiant-secret-shop",
  "mapVersion": "7.xx",
  "answer": { "x": 0.25, "y": 0.7 },
  "cropCenter": { "x": 0.5, "y": 0.5 },
  "difficulties": ["easy", "medium", "hard"],
  "credit": "GitHubUsername"
}
```

Coordinates are normalized from `0` to `1`, measured from the top-left corner.
The `cropCenter` field is optional and defaults to the image center.

Run `pnpm validate:locations` before opening a pull request.
